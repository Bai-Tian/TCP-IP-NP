# 基于TCP的服务器端/客户端（1）

TCP/IP协议栈共分4层

```
+--------------------+
|     Application    |
+--------------------+
|       TCP/UDP      |
+--------------------+
|         IP         |
+--------------------+
|      Ethernet      |
+--------------------+
```

### TCP服务器端默认函数调用顺序

```
+------------------+
|      socket()    |创建套接字
+------------------+
|       bind()     |分配套接字地址
+------------------+
|      listen()    |监听连接
+------------------+
|      aceept()    |允许连接
+------------------+
|  read()/write()  |数据交换
+------------------+
|      close()     |断开连接
+------------------+
```

### TCP客户端默认函数调用顺序

```
+------------------+
|      socket()    |创建套接字
+------------------+
|      connect()   |请求连接
+------------------+
|  read()/write()  |数据交换
+------------------+
|      close()     |断开连接
+------------------+
```
